GoIT GoJava#3 ментор Шкурупий Дмитрий

Данный проект представляет собой Final Core Project группы Шкурупия Дмитрия.

Интерфейсная часть представляет собой экзекьютор выполненных проектов.

Для успешной интеграции проекта требуется подготовить:
1. Описание проекта. Что, как, почему и зачем применялось.
2. UML-диаграммы в виде image-файлов (*.jpg)
3. Раннер проекта.
4. Тесты.

Инструкция по дабавлению проектов:
1. Загрузить репозитарий 'https://github.com/Dmitrij0/FinalCoreFXProjects.git'

2. Добавить свои исходники и тесты
2.1. Исходники следует помещать в пакет 'com.gojavaonline3.shkurupiy.finalcore.XXXXXXX',
     где 'XXXXXXX' - фамилия автора.
2.2. Исходные файлы проекта '/src/main'
2.3. Исходные файлы тестов '/src/test'
2.4. Файлы с UML '/src/main/resources/XXXXXXX',
     где 'XXXXXXX' - фамилия автора.
2.5. Не забыть исправить имена пакетов

3. Добавить свои проекты в конфигурационный файл '/src/main/resources/config.xml' по образцу.

4. Закоммитить изменения и выложить в удаленный репозитарий.
